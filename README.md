# The Daily Rhythm

Family discipleship & schedule app for Jaxson, Mikiah, and Kennedy.

## Local development
```
npm install
npm run dev
```
Open http://localhost:5173

## Deploy to Vercel (free)
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → import the repo
3. Click Deploy — done. Vercel auto-detects Vite.

## Notes
- Parent PIN default: 1234 (change inside the app under Manage → Settings)
- All data saves to browser localStorage — works on any device
- For shared data across devices, a future upgrade can add Supabase
